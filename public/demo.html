Browser verifier
Open darkmatterhub.ai/verify, drop the bundle. Runs in your browser — file never leaves your machine.

Python script
Run the verifier locally. Works offline.
python verify_darkmatter_chain.py proof_bundle.json

Any cryptography library
The proof bundle contains an Ed25519 signature and SHA-256 hash chain. Any standard cryptography tool can verify them — no DarkMatter software required.
