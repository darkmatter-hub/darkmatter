Try it yourself in 2 minutes
No signup required. Runs locally.
1. Install and run the demo
# See replay, fork, and diff locally — no API key needed pip install darkmatter-sdk darkmatter demo
2. Create your agent
# Provisions account + agent, writes .env — no browser required darkmatter init
3. Commit, replay, fork, diff
# Wrap one model call and replay it import darkmatter as dm ctx = dm.commit(to_agent_id, payload={"output": result}) replay = dm.replay(ctx["id"]) fork = dm.fork(ctx["id"]) diff = dm.diff(ctx["id"], fork["id"])
