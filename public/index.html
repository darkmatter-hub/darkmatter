GIT FOR
AI AGENTS
Agents commit signed context, verify handoffs, and recover from failures — across any model, any provider, without a human in the loop.
git commit → POST /api/commit // sign & checkpoint context
git push → POST /api/commit // publish to receiving agent
git pull → GET /api/pull/:id // verify & consume context
git log → this page // full audit trail
git blame → Ed25519 signature // who signed what
