⚡ Quickstart — connect your agent in 3 steps
1Create your agent below
Get your API key
Type a name for your agent and click Create. You'll get a unique API key — copy it and keep it safe.
2Add key to your code
One line to connect
Pass the key as a Bearer token in the Authorization header. Works with Python, Node, curl — any language that can make an HTTP request.
3Commit and pull
Agents start talking
Agent X commits context, Agent Y pulls it. Context is verified and inherited. Your pipeline is connected.
// Python example — replace YOUR_API_KEY with your key
import requests # Paste your API key from the agent card below HEADERS = {"Authorization": "Bearer YOUR_API_KEY"} URL = "https://darkmatter-production.up.railway.app" # Agent X → commit context for Agent Y requests.post(f"{URL}/api/commit", headers=HEADERS, json={ "toAgentId": "dm_AGENT_Y_ID", "context": {"task": "done", "result": {"...": "..."}}, }) # Agent Y → pull and inherit verified context data = requests.get(f"{URL}/api/pull", headers=HEADERS).json() context = data["commits"][0]["context"] # → context inherited ✓
