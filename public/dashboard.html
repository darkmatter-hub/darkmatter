Claude Managed Agents proxy
NEW
Record every Claude API call automatically — including Managed Agent sessions, messages, and events. Two setup options.
Option 1 — Environment variable (fastest)
# In your shell or .env file
export ANTHROPIC_BASE_URL=https://darkmatterhub.ai/ext/claude
export ANTHROPIC_API_KEY=sk-ant-... # your real key
# Add your DarkMatter proxy key
export DM_PROXY_KEY=loading...
Then use the Anthropic SDK normally — all calls are automatically recorded.
export ANTHROPIC_BASE_URL=https://darkmatterhub.ai/ext/claude
Option 2 — Dashboard key (no env vars)
Add your Anthropic key in Settings → Claude Recording. DarkMatter will use it automatically for all proxy calls.
✓ Works with Messages API, Managed Agents sessions, streaming, and all Claude models
✓ Your API key is never logged — forwarded directly to Anthropic
✓ Every call becomes a verifiable DarkMatter record
